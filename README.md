# User Authentication System

## Task 1 - Prodigy InfoTech Full Stack Web Development Internship

### Features
- User Registration
- User Login
- User Logout
- Password Hashing using Django Authentication
- Session Management
- Protected Routes
- Responsive Bootstrap UI

### Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap 5
- SQLite

### How to Run

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open:

http://127.0.0.1:8000/
